## Example
Include the icon using the component name and add any properties or [common HTML attributes](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/svg) you would to any SVG Element.

```javascript
import { IconQuickShip } from 'components/icons';

return(
    <IconQuickShip 
        width="50" 
        height="50" 
        className={Style.icon} 
    />
)

```
